# 🎤 Mental Health Voice AI Assistant

A real-time voice-based AI assistant that allows users to express their emotions through speech and receive supportive, human-like responses in voice.

---

## 💡 Project Overview

This project focuses on building a **voice-first mental health assistant** where users can:

- Speak naturally about their feelings 🎤  
- Get emotionally supportive AI responses 💬  
- Hear responses in voice 🔊  

It demonstrates how AI can make mental health support more **accessible, natural, and engaging**.

---

## 🚀 Features

- 🎤 Real-time voice input (microphone)
- 🧠 Dynamic AI responses (no keyword-based logic)
- 💬 Human-like emotional understanding
- 🔊 Voice-based output using Text-to-Speech
- ⚡ Fast processing with modern AI APIs
- 🎨 Simple and interactive UI

---

## 🛠️ Tech Stack

| Technology | Usage |
|----------|------|
| HTML, CSS, JS | Frontend UI |
| Flask (Python) | Backend server |
| Whisper | Speech-to-Text |
| Groq API | AI Response Generation |
| Murf API | Text-to-Speech |

---

## 🔄 How It Works

1. User speaks through microphone 🎤  
2. Audio is sent to Flask backend  
3. Whisper converts speech → text  
4. AI model generates emotional response  
5. Murf converts text → voice  
6. Audio is sent back and played 🔊
